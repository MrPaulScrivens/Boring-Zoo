## Psychology Through the World Code Lens

The World Code IS psychology applied to business. Each element maps to fundamental psychological principles:

### Climax = Loss Aversion + Contrast Effect
- **Before State** triggers loss aversion — they're losing time, money, opportunities
- **After State** creates contrast — the gap between where they are and where they could be
- **Transformation Promise** = the relief of resolving that tension

### Method = Authority + IKEA Effect
- **Why It Works Differently** establishes authority — you've found a better way
- **Method Phases** create the IKEA Effect — as they invest effort, they value the transformation more
- **Method Name** becomes a mental anchor — a shortcut for "the right approach"

### Conversation = Mere Exposure + Confirmation Bias
- **Content Themes** create mere exposure — repeated contact builds familiarity
- **Wrong Belief** leverages confirmation bias — people who already suspect the Wrong Belief will be drawn to your message
- **Core Message** becomes a mantra — repetition creates belief

### Crossing = Commitment/Consistency + Reciprocity
- **Entry Point** uses foot-in-the-door — small first commitment leads to bigger ones
- **Readiness Signals** identify people with high motivation (BJ Fogg model)
- **Buying Experience** reduces activation energy

### Creation = Endowment Effect + Anchoring
- **BONES breakdown** uses anchoring — stack value before revealing price
- **Pricing** leverages mental accounting — frame against the cost of the Before State

### Voice = Liking/Similarity + Unity
- **Authenticity Markers** trigger the Liking bias — people trust those who seem real
- **Shared language** creates Unity — "one of us" feeling

## Foundational Thinking Models

### First Principles (World Code Style)
Don't copy competitors' marketing. Ask: "What does my World Code say the answer should be?" Your Method is your first principle.

### Jobs to Be Done
People "hire" your Creation to get from Before State to After State. Frame everything around this job.

### Inversion
Instead of "How do I convert more people?", ask "What would guarantee nobody buys?" Then check your marketing for those anti-patterns.

### Pareto Principle
Which 20% of your Conversation themes drive 80% of engagement? Focus there.

## Buyer Psychology Models

### Loss Aversion / Prospect Theory
People avoid loss 2x more than they pursue gain.
**World Code application**: Frame the Before State as ongoing loss. "Every day in [Before State], you're losing [specific thing]."

### Social Proof / Bandwagon Effect
People follow what others do.
**World Code application**: Show others who've completed the Climax transformation. Numbers, stories, testimonials — all through your Voice.

### Anchoring Effect
First numbers heavily influence judgment.
**World Code application**: Anchor to the cost of the Before State, then reveal your Creation price. The contrast makes it feel small.

### Endowment Effect
People value what they own more.
**World Code application**: Free Method Phase 1 lets them "own" progress. They won't want to give it up.

### Status-Quo Bias
People prefer the current state.
**World Code application**: Challenge the Wrong Belief (from Conversation) to shake people from status quo. Make staying in the Before State feel more dangerous than changing.

### Confirmation Bias
People seek information confirming existing beliefs.
**World Code application**: Your Conversation Core Message should align with what your audience SUSPECTS is true but hasn't confirmed. Validate their instinct.

## Persuasion Models

### Reciprocity
Give first, people give back.
**World Code application**: Conversation theme content = generous giving. Method Phase 1 free = reciprocity trigger.

### Commitment & Consistency
Small commitments lead to bigger ones.
**World Code application**: The Crossing journey IS progressive commitment — Entry Point → deeper engagement → Creation purchase.

### Scarcity
Limited availability increases perceived value.
**World Code application**: Only use when genuine. Limited cohorts for Method workshops, closing enrollment for Creation.

### Authority
People defer to expertise.
**World Code application**: Your Method IS your authority claim. "Why It Works Differently" positions you as the expert.

## Pricing Psychology (World Code)

### Anchoring
Show the cost of the Before State first, then your Creation price.

### Framing
"$X/month" vs "$X/day" vs "Less than [common Before State expense]"

### BONES as Value Stack
Each BONES element adds perceived value before revealing price.
