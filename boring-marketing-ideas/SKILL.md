## The World Code Filter

Before recommending any idea, run it through this filter:

1. **Voice Check** — Can this be done in their authentic Voice? If it requires sounding generic, skip it.
2. **Method Alignment** — Does this reinforce their unique Method? Or does it look like everyone else?
3. **Climax Connection** — Does this speak to people in the Before State and promise the transformation?
4. **Conversation Fit** — Does this fit within their Content Themes? Or is it off-topic?
5. **Crossing Match** — Does this use their Discovery Channel? Or force a channel they don't own?

If an idea fails 2+ checks, deprioritize it. The best marketing reinforces the World Code at every touch.
