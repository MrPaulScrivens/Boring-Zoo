## The World Code Referral Loop

```
Transformation Moment → Share Impulse → Refer Someone in Before State → They Experience Method → Transformation → (Loop)
```

The best referrals happen naturally when someone experiences the Climax transformation. Your job is to make sharing easy at that moment.

## Referral Program Design

### Step 1: Identify Trigger Moments (World Code Specific)

**High-intent moments tied to your World Code:**
- Right after experiencing the Climax transformation
- After completing a Method phase milestone
- After a breakthrough that confirms the Core Message
- After engaging deeply with a Conversation theme

### Step 2: Design Share Mechanism

**Ranked by effectiveness:**
1. Direct sharing (highest conversion)
2. Personalized referral link
3. Email invitation
4. Social sharing
5. Referral code

### Step 3: Choose Incentive Structure

**Align incentives with your Creation:**
- Offer Creation-related rewards (access to bonus Method content, extended access, etc.)
- Double-sided rewards work best (both parties get something aligned with the transformation)
- Tiered rewards for repeat referrers (deeper access to Method resources)

## Referral Messaging — In Voice

All referral messaging must use the user's Voice from `voice.md`.

### Referral Invite Copy

Instead of generic "Share with a friend," use World Code language:
- "Know someone stuck in [Before State]? Share [Method Name] with them."
- "The [Climax transformation] shouldn't stay a secret."
- "[Creation Name] changed your [Before State problem]. Pass it on."

### Landing Page for Referred Users

Structure:
1. **Who referred them** — Personal endorsement
2. **Before State recognition** — "If you're dealing with [Before State]..."
3. **Method introduction** — Brief intro to the approach
4. **Climax promise** — The transformation that awaits
5. **Entry Point CTA** — From `crossing.md`

## Program Optimization

### Improving Referral Rate

**If few people are referring:**
- Ask at transformation moments, not random moments
- Make sharing feel natural to your Voice (not corporate)
- Test incentives aligned with Creation value

**If referrals aren't converting:**
- Improve the referred user experience (walk them into the World)
- Strengthen the Before State recognition on landing pages
- Ensure the referrer's endorsement is visible

### A/B Tests to Run

- **Messaging tests**: Different Climax angles in referral copy
- **Timing tests**: Which Method milestone triggers best referrals
- **Incentive tests**: Creation-aligned rewards vs. generic

---

## Measuring Success

### Key Metrics

**Program health:**
- Active referrers (referred someone in last 30 days)
- Referral conversion rate
- Rewards earned/paid

**World Code impact:**
- % of new people from referrals (vs. Crossing Discovery Channel)
- Do referred users complete more Method phases?
- Do referred users have higher transformation rates?
