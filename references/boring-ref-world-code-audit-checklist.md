# World Code Audit Checklist

Review any marketing asset against the six World Code elements. Use the Quick Audit for a rapid pass. Use the Deep Audit when you need to diagnose exactly where alignment breaks down.

**The six World Code elements:**
- **Voice** — Tone, rhythm, hard rules, authenticity markers, sentence structure, vocabulary, opening style
- **Climax** — Transformation promise, who it's for, before state, after state, why it matters
- **Method** — Methodology name, core principle, phases, why it works differently
- **Creation** — Offer name, format, support level, BONES (Big, Obvious, New, Easy, Safe), program structure, pricing
- **Conversation** — Core message, wrong belief, content themes, platform/rhythm, communication style
- **Crossing** — Discovery channel, entry point, readiness signals, buying experience, real objection, invitation language

---

## Quick Audit (5-Minute Check)

A rapid pass for any marketing asset. If any box stays unchecked, dig into the Deep Audit for that element.

- [ ] Does it sound like the Voice? (Read it aloud — does it feel right?)
- [ ] Is the Climax clear? (Can the reader picture the transformation?)
- [ ] Is the Method referenced or implied? (Does it feel unique, not generic?)
- [ ] Does it connect to the Creation? (Is there a path to the offer?)
- [ ] Does it serve the Conversation? (Does it reinforce the core message?)
- [ ] Does it support the Crossing? (Does it move someone closer to buying?)

---

## Deep Audit by Element

### Voice Audit

- [ ] Tone matches Voice profile (energy, attitude, warmth, formality)
- [ ] Sentence structure follows Voice rhythm patterns
- [ ] No Hard Rule violations (check every banned word/phrase)
- [ ] Vocabulary level is consistent with Voice
- [ ] Opening style matches Voice opening pattern
- [ ] Authenticity markers are present
- [ ] No generic marketing language that contradicts Voice
- Score: ___/7

### Climax Audit

- [ ] Transformation Promise is stated or clearly implied
- [ ] Before State is painted (reader feels understood)
- [ ] After State is vivid and specific (reader can picture it)
- [ ] The right person would self-identify ("this is for me")
- [ ] The wrong person would self-select out
- [ ] Deeper significance (identity shift) is present
- Score: ___/6

### Method Audit

- [ ] Methodology Name is used (if appropriate for the asset type)
- [ ] Core Principle is communicated or implied
- [ ] Approach feels different from conventional wisdom
- [ ] "Why It Works Differently" is clear
- [ ] Phases are referenced (if appropriate for the asset type)
- Score: ___/5

### Creation Audit (for sales/offer assets)

- [ ] Offer Name is used
- [ ] Big — the problem solved feels significant
- [ ] Obvious — the outcome is crystal clear
- [ ] New — it feels different from what they've tried
- [ ] Easy — the path to results feels achievable
- [ ] Safe — risk is addressed or eliminated
- [ ] Pricing is justified (if shown)
- [ ] Program structure is clear
- Score: ___/8

### Conversation Audit (for content assets)

- [ ] Reinforces the Core Message
- [ ] Addresses or dismantles the Wrong Belief
- [ ] Maps to a Content Theme
- [ ] Matches the Communication Style
- [ ] Would sustain interest over 2+ years of similar content
- Score: ___/5

### Crossing Audit (for conversion assets)

- [ ] Discovery channel is appropriate
- [ ] Entry point is clear (if applicable)
- [ ] Readiness signals are acknowledged
- [ ] Buying experience matches the defined approach
- [ ] Real objection is addressed (not avoided)
- [ ] Invitation language is used (not pushy sales tactics)
- [ ] Feels like an invitation, not a pitch
- Score: ___/7

## Audit by Asset Type

Which elements to prioritize for each asset.

### Social Media Post
Primary: Voice, Conversation, Climax
Secondary: Method (if teaching), Crossing (if CTA)

### Email
Primary: Voice, depends on email type
- Welcome email: Climax, Method
- Content email: Conversation
- Sales email: Creation, Crossing

### Sales Page
Primary: ALL elements — full audit required

### Landing Page (lead magnet)
Primary: Voice, Climax, Crossing
Secondary: Method (tease)

### Ad Copy
Primary: Voice, Climax, Method (differentiation)
Secondary: Crossing (CTA)

### Blog Post / Article
Primary: Voice, Conversation, Method
Secondary: Climax (context), Crossing (CTA if applicable)

## Scoring Guide

- **90-100%:** Strong World Code alignment. Ship it.
- **70-89%:** Good but has gaps — identify and fix the weak elements before publishing.
- **50-69%:** Significant misalignment — major revision needed. Revisit the World Code elements that scored lowest.
- **Below 50%:** Start over with your World Code elements in front of you.

---

## Common Misalignment Patterns

1. **"Voice drift"** — starts in Voice but slides into generic marketing language by the middle or end
2. **"Climax confusion"** — vague transformation, no concrete After State, reader can't picture what changes
3. **"Method invisibility"** — sounds like everyone else, no unique methodology present, interchangeable advice
4. **"BONES gaps"** — offer missing one or more BONES elements, usually Safe or New
5. **"Wrong belief blindness"** — content doesn't fight against anything, no tension, no reason to change
6. **"Pushy crossing"** — sales language that contradicts the invitation philosophy, urgency tactics that feel manipulative
