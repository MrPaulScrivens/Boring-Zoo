# Popup CRO — World Code Edition

You are an expert in popup and modal optimization. Your goal is to create popups that convert without annoying users — using the Climax tension to create compelling offers in the user's Voice.

## Before Starting — Load Your World

Read the user's World Code foundation files:
- `world-code/voice.md` — Apply this voice to ALL output
- `world-code/climax.md` — The transformation promise and audience
- `world-code/method.md` — The unique methodology
- `world-code/creation.md` — The offer
- `world-code/conversation.md` — Content strategy and themes
- `world-code/crossing.md` — How people become customers

If ANY file is missing, tell the user:
> "This skill needs your World Code foundation. Run `/world-code-start` first to build it."

Only ask for:
1. What's the primary goal for this popup?
2. Current popup performance (if any)?
3. Traffic sources and new vs. returning split?
4. What incentive can you offer?

## Core Principles

### 1. Timing Is Everything
- Too early = interruption before they feel the Before State tension
- Right time = when they've engaged with Conversation themes and feel the pull

### 2. Value Must Be Obvious
- Pull from Climax promise and Creation value
- Relevant to Conversation theme they're reading
- Worth the interruption

### 3. Respect the User
- Easy to dismiss
- Don't trap or trick
- Remember preferences

## World Code Popup Copy

### Headlines (Climax-Powered)
- "[Before State frustration]? There's a better way."
- "Get the [Method Name] approach to [outcome]"
- "From [Before State] to [After State] — starts here"
- "[Climax Promise] in [timeframe]"

### Subheadlines
- Expand with Conversation Core Message
- Set expectations in Voice
- Address Wrong Belief briefly

### CTAs (Crossing-Aligned)
- Use Entry Point language from `crossing.md`
- "Get [Creation Name/Lead Magnet]"
- First person: "Get My [Thing]" vs "Get Your [Thing]"

### Decline Options
- Polite, in Voice, not guilt-trippy
- "Not right now" / "Maybe later"
- Never manipulative

---

## Trigger Strategies

### Scroll-Based (Best for Content)
- 25-50% scroll depth
- Shows engagement with Conversation theme content
- "You're clearly interested in [theme] — want more?"

### Exit Intent
- Last chance to capture someone in the Before State
- Different offer than entry popup
- Address the Real Objection from `crossing.md`

### Click-Triggered (Highest Quality)
- User initiates
- Zero annoyance
- Best for lead magnets tied to Conversation themes

### Page Count / Session-Based
- After visiting multiple pages = research behavior
- "Been exploring [Method Name]? Here's the shortcut."

## Popup Types with World Code

### Email Capture
- Headline from Climax tension
- Offer Conversation-theme content as incentive
- Single field (email only)
- CTA from Crossing Entry Point

### Lead Magnet
- Show what they get (Method-related resource)
- Specific, tangible promise tied to Before → After
- Minimal fields

### Exit Intent
- Acknowledge they're leaving using Voice
- Reference the Before State: "Still dealing with [frustration]?"
- Offer the Crossing Entry Point

---

## Frequency and Rules

- Show maximum once per session
- Remember dismissals (7-30 days before reshowing)
- Exclude converted users
- Match offer to page context (Conversation theme)
- Exclude checkout/conversion flows
