# Boring Zoo Skills

Claude Code skills that help you build a business around your worldview.

These skills use the **World Code** framework to ground every marketing decision — copy, CRO, SEO, sales, growth, and strategy — in your unique worldview rather than generic best practices.

## Install

Install all skills at once:

```bash
npx skills add MrPaulScrivens/Boring-Zoo
```

Install a single skill:

```bash
npx skills add MrPaulScrivens/Boring-Zoo --skill boring-copywriting
```

## Skills

### Content
- **boring-copywriting** — Write marketing copy using your voice, climax, method, and crossing
- **boring-copy-editing** — Edit existing copy against your voice rules and World Code consistency
- **boring-content-strategy** — Plan content strategy with your pillars, themes, and audience
- **boring-social-content** — Create social media content with your content pillars and hooks
- **boring-marketing-ideas** — Generate marketing ideas filtered through your strategy
- **boring-seo-article** — Write SEO-optimized articles from research to final score

### CRO
- **boring-page-cro** — Optimize pages against your Climax promise and Crossing journey
- **boring-form-cro** — Optimize any form for conversion
- **boring-onboarding-cro** — Optimize post-signup onboarding and activation
- **boring-signup-flow-cro** — Optimize signup and registration flows
- **boring-popup-cro** — Create or optimize popups and overlays
- **boring-paywall-upgrade-cro** — Optimize upgrade screens and paywall experiences
- **boring-ab-test-setup** — Plan and implement A/B tests grounded in World Code

### SEO
- **boring-ai-seo** — Optimize for AI search engines and LLM citations
- **boring-programmatic-seo** — Create SEO pages at scale with templates
- **boring-schema-markup** — Add or fix structured data and schema markup
- **boring-seo-audit** — Audit and diagnose SEO issues
- **boring-site-architecture** — Plan URL structure, navigation, and internal linking

### Sales
- **boring-pricing-strategy** — Pricing, packaging, and monetization strategy
- **boring-sales-enablement** — Pitch decks, one-pagers, objection handling, demo scripts
- **boring-revops** — Revenue operations and lead lifecycle management
- **boring-cold-email** — Cold emails and follow-up sequences

### Growth
- **boring-referral-program** — Create or optimize referral and affiliate programs
- **boring-free-tool-strategy** — Plan free tools for marketing purposes
- **boring-launch-strategy** — Plan product launches and announcements
- **boring-paid-ads** — Paid ad campaigns across platforms
- **boring-ad-creative** — Generate and iterate ad creative

### Strategy
- **boring-marketing-psychology** — Apply behavioral science to marketing
- **boring-product-marketing-context** — Generate a World Code Summary for other tools
- **boring-competitor-alternatives** — Competitor comparison pages
- **boring-analytics-tracking** — Set up and audit analytics tracking
- **boring-human** — Humanize AI-generated content
- **boring-remix** — Remix existing content into new formats

### Retention
- **boring-churn-prevention** — Reduce churn with cancellation flows and save offers
- **boring-email-sequence** — Create email sequences and lifecycle campaigns

### World Code Foundation
- **world-code-start** — Initialize your World Code
- **world-climax** — Define your transformation promise
- **world-conversation** — Define your content themes
- **world-creation** — Define your product/offer
- **world-crossing** — Define your customer journey
- **world-crown** — Define your audience
- **world-method** — Define your unique approach
- **world-voice** — Define your brand voice
